Multilingual Group Chat Web App

This is a real-time group chat web application that enables users to communicate in their native languages with the help of automatic language translation. It uses the Google Translate API to translate each user's messages into the preferred language of every other member in the group. The platform supports user registration, login, group creation, member management, and real-time multilingual chatting. The backend is developed using Flask with SQLite for data storage, while the frontend is built with HTML, CSS, and JavaScript. This project focuses on seamless communication, translation accuracy, and a user-friendly interface—encouraging collaboration across language barriers in teams, classrooms, or global communities.

Food Freshness Detection Using Machine Learning

This project is designed to promote food safety by identifying fresh and spoiled fruits in an image. By simply uploading a fruit image, the system automatically analyzes each fruit's type, condition (fresh or spoiled), and provides a count for both categories. The core of this solution is powered by the YOLOv8 (You Only Look Once version 8) object detection model, which is capable of detecting multiple fruits in a single image with high accuracy and speed. The model was trained on a custom dataset containing various fruits in both fresh and rotten conditions. This system is useful in reducing food waste, improving quality checks in markets and stores, and helping consumers make safer food.
